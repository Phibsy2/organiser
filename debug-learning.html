Learning Module Debug

1. Gehe zu JuraFit App

Öffne die App und gehe zum "Lernen" Tab

2. Öffne Browser DevTools

Drücke F12 oder Rechtsklick → "Element untersuchen"

3. Gehe zur Console

Klicke auf den "Console" Tab in den DevTools

4. Klicke auf eine Lerneinheit

Du solltest folgende Logs sehen:

  • Unit clicked: [unit-id]
  • Is enrolled: true/false
  • handleUnitSelect called with: [unit-id]
  • Available modules: [array]
  • Found unit: [unit-title]
  • Current view: unit
  • Rendering LearningUnit with: [unit-title]

5. Was passiert?

Teile mit mir die Console-Ausgabe, dann kann ich das Problem identifizieren.
